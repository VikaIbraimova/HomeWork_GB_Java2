1)MyExampleSingleton - показывает пример написания паттерна проектирования Singleton.
Данный пример применим для однопоточных приложений.
2)MyExampleSingleton2 - вариант для многопоточного приложения.Но пока не скидывала,сырой.
3)MyExampleFactoryMethod - показывает пример написания паттерна проектирования FactoryMethod.
Пояснение:
1)Есть интерфейс Car c двумя объявленными в нем методами: drive(),stop(). 
2)Есть классы, реализующие этот интерфейс: Jeep,Porshe,NewJeep.
3)Есть перечисдение RoadType(список дорог).
Создается специальный метод getCar, в отдельном классе CarSelector(данный класс принято называть фабрикой).
В зависимости от входящих условий(элемент списка RoadType) метод getCar создает объект класса Car:Porshe,Jeep,NewJeep.
